## ğŸ“ Customization Guide

### Updating Content
- Edit the HTML files directly to change text, images, and content
- All pages follow a consistent structure for easy updates

### Adding Images
- Replace placeholder emoji icons with actual images
- Update image paths in the HTML files
- Recommended: Create an `assets/images/` folder for images

### Integrating Payment
- Update donation buttons in `get-involved.html` with your PayFast/PayPal integration
- Replace placeholder payment forms with actual payment gateway code

### Adding Google Maps
- In `contact.html`, replace the placeholder with your Google Maps embed code
- Get embed code from Google Maps: Right-click on your location â†’ "Share or embed map"

### Adding YouTube Videos
- In `gallery.html` and `dream-stories.html`, replace placeholders with YouTube embed code
- Format: `<iframe src="YOUR_YOUTUBE_EMBED_URL"></iframe>`

### Social Media Links
- Update social media icons in the footer with your actual links
- Replace placeholder emoji icons with actual social media icons or links
